Thinking like a Programmer

If you want to "think like a programmer", practice the following ways of thinking; procedural thinking, abstract thinking, systematic debugging, systematic thinking, and technological empathy.

It is important to avoid repetition when writing code so that your code does not become cluttered and so that you use your time in the most efficent way possible.

03

How to add CSS to your Web Page:

  1. Write CSS in the <head> of your HTML
  2. Link your HTML to a seperate CSS file. <link rel="stylesheet" type="text/css" href="style.css>
  3. Write your style inline with the HTML.

HTML

<p> This is an example of corresponding HTML & CSS code side-by-side.</p>


CSS

     }
Margin the space between boxes.

Padding clears an area around the content.

Content the image or text.

Inheritance A mechanism by which properties are applied not only to a specific element, but, also to its descendants. It relies on the document tree.
